# CLEAR-LeWM v0.3 manifests

The eight manifests in this directory freeze the exact Moderate and Strict
pair IDs used by the v0.3 audit. They are organized as:

```text
manifests/v0.3/<task>/<mode>-seed42-n100.json
```

All selected pairs are initially unsolved under their embedded rollout
predicate. TwoRoom manifests contain 100 cross-room goals with full-disk clear
starts and goals. Do not edit a manifest after publishing a result that cites
its SHA-256.
