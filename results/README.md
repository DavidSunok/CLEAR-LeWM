# Results

`reference/` contains compact baseline outputs generated by the public runner.
Large videos, model checkpoints, and local experiment outputs are intentionally
ignored by Git.

Each result records its manifest SHA256, policy seed, criterion, solver
configuration, package versions, aggregate metrics, and per-episode outcomes.

## CLEAR-ID deterministic random baselines

| Task | Episodes | Random SR |
|---|---:|---:|
| PushT | 100 | 3% |
| Reacher | 100 | 9% |
| TwoRoom | 100 | 22% |
| OGBench-Cube | 100 | 1% |

These are single, fixed-manifest reference baselines rather than estimates over
training seeds. Their purpose is to make each task's policy-independent floor
visible and provide paired outcomes for method comparisons.
